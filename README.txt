Название команды:
    VRhome

Описание идеи:
    Комнаты для визуализации идей, медитации и просмотра фильмов виртуальной реальности в WebVR: 
        - Просматривай свои недавние фотографии и видео прямо на Тайм-Сквер 
        - Отслеживай страны в которых ты побывал на корабле прямо как Колумб 
        - Выполняй вход в вк, фейсбук, твиттер и т.д. через двери в открытый космос 
        - Создай комнату для медитации 
        - Погружайся в планирование своего будущего с помощью комнаты для визуализации 
        - Черпай вдохновение для проектов собирая все необходимые материалы в VR и проводи там столько времени,
        сколько нужно для появления новой классной идеи! 
        
        А все потому что VRhome - твой новый дом.
